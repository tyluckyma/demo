Selected Research Projects

Public Transit Performance Analysis using Smart Card Data (Hong Kong MTR)

Analyzed anonymized Octopus card transaction data to evaluate MTR network performance. Developed Python scripts (Pandas, GeoPandas) to calculate passenger flow metrics, identify peak-hour congestion points, and model transfer efficiency between lines. Visualized results using QGIS and Matplotlib.

Technologies/Methodologies: PythonPandasGeoPandasQGISNetwork AnalysisBig Data Analytics

Key Findings: Identified specific station pairs with high transfer friction and proposed optimized signage/layout recommendations based on simulated passenger flow improvements.

View Project Report Snippet

Urban Heat Island Mitigation Potential Mapping (Shenzhen)

Utilized Landsat 8/9 thermal bands and Sentinel-2 multispectral imagery to map Land Surface Temperature (LST) variations across Shenzhen. Correlated LST with high-resolution land use/land cover data and building morphology metrics derived using ArcGIS Pro and Google Earth Engine. Modeled the cooling potential of proposed green infrastructure interventions.

Technologies/Methodologies: Remote SensingArcGIS ProGoogle Earth EngineRaster AnalysisSpatial Statistics (GWR)Environmental Modeling

Key Findings: Produced high-resolution maps identifying priority areas for UHI mitigation efforts, quantifying the expected temperature reduction from targeted greening strategies.
